So verwenden Sie den KI-Unterschriftgenerator


Stiftfarbe

Klicken Sie auf das Stiftsymbol in der Toolbox, um die Stiftfarbe anzupassen. Ein Popup wird angezeigt, in dem Sie eine Farbe aus einer Farbauswahl auswählen können.

Leinwand anpassen

Verwenden Sie das zweite Symbol, um die Leinwandabmessungen zu ändern. Sie können die Größe des Unterschriftenpads nach Ihren Wünschen anpassen.

Hintergrundfarbe der Leinwand

Klicken Sie auf das Hintergrundfarbensymbol, um den Leinwandhintergrund zu ändern. Über eine Farbauswahl können Sie eine neue Farbe auswählen.

Leinwand leeren

Wenn Sie von vorne beginnen müssen, klicken Sie auf das Papierkorbsymbol, um die aktuelle Leinwand zu leeren und eine neue Signatur zu beginnen.

Herunterladen

Wenn Sie mit Ihrer Signatur zufrieden sind, klicken Sie auf das Download-Symbol, um Ihr Signaturbild zu generieren und zu speichern.
